### 1.2 变量的数据类型

变量在赋值时会自动判断数据的类型

> Python最常用的四种数据类型

| 类型                   | 示例           |
| ---------------------- | -------------- |
| 字符串 - str(string)   | name = "Jack"  |
| 整型 - int(integer)    | age = 20       |
| 浮点型 - float         | weight = 148.6 |
| 布尔型 - bool(boolean) | is_true = True |



> type函数

- type函数用于得到变量的数据类型
- 语法：变量 = type(变量名)
- 输出：str | int | float | bool
